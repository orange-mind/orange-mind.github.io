# OrangeMindHelper

OrangeMindHelper是OrangMind的辅助扩展，该扩展仅支持注入页面脚本，不上报任何用户信息，请放心使用。由于Chrome v3版本扩展的限制，请按照如下流程安装该扩展。

  1. 下载 OrangeMindHelper (opens new window) 扩展并解压。
  2. 打开 chrome://extensions/ 页面，点击开启右上角的开发者模式
3. 如上图，点击页面左上角 `加载已解压的扩展程序` 按钮，选中刚下载的OrangeMindHelper目录。
