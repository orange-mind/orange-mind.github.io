# OrangeMind

为了高效使用Web页面，减少上下文切换和打断，随时随地在网页执行快捷操作。

我花了3年时间，开发了一款Chrome扩展：OrangeMind (opens new window)。

OrangeMind包含快捷指令、Web编辑器、页面脚本和代码片段共四大核心功能。

# 规划

除了上述这些功能外，OrangeMind还有更多强大功能正在快速开发调试中，敬请期待~。使用过程中有任何问题和建议，欢迎在评论区或 github (opens new window) 留言，好的想法也会加入后续迭代计划。
