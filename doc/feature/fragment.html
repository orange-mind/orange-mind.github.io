# 编辑

点击编辑按钮，打开代码片段编辑弹框如下所示：

以上编辑弹框字段含义如下所示：

  • 语言：表示在当前代码语言下会出现输入提示
  • 关键词：表示输入触发提示的词语
  • 输入说明：表示输入提示的说明
  • 代码片段：表示选中该条代码片段提示时，实际插入的代码
